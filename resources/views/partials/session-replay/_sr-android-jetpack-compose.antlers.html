{{markdown}} ## Jetpack Compose Capturing `@Composable` Compose views requires extra setup. Add the Compose UI Tooling library to your project. ```kotlin implementation("androidx.compose.ui:ui-tooling:${composeVersion}") ``` Session Replay for Android supports Jetpack Compose `1.5.x` and `1.6.x`. It's tested on `[1.5.3,1.6.4]`. {{/markdown}}
