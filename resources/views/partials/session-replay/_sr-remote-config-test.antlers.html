{{markdown}} ### Remote configuration Enable remote configuration to set Sample Rate and Masking Level in Amplitude. {{partial:admonition type="note" heading="Remote configuration and testing"}} {{markdown}} With `enableRemoteConfig` set to `true`, settings you define in Amplitude take precedence over settings you define locally in the SDK. For this reason, while testing your application, you should **disable** remote configuration to ensure you can set `sampleRate` to `1`, and ensure you capture test sessions. {{/markdown}} {{/partial:admonition}} {{/markdown}}
